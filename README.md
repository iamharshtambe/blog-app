# Friend's Blog

A simple React blog app to create, view, and delete posts.

## Setup

1. Install dependencies

```bash
npm install
```

2. Install JSON Server

```bash
npm install -g json-server
```

3. Start backend

```bash
json-server --watch db.json --port 8000
```

4. Start app

```bash
npm run dev
```

## Features

-  Create blog posts
-  View all posts
-  Delete posts
-  Simple, clean UI
